# Installation

Prometheux can be deployed in multiple ways depending on your infrastructure
and requirements. Choose the option that best fits your environment.

| Method | Best For | Infrastructure | Effort |
| --- | --- | --- | --- |
| [**Prometheux Cloud**](/getting-started/installation/cloud-platform) | Fastest start, no infra to manage | None (hosted by Prometheux) | Minimal |
| [**On (Cloud) Premises**](/getting-started/installation/on-cloud-premises) | Own VM or cluster on AWS, Azure, GCP, or bare metal | Docker / Yarn / Kubernetes | Low–High |
| [**Snowflake Native App**](/getting-started/installation/snowflake-native-app) | Teams already on Snowflake | Snowflake account | Low |
| [**Databricks**](/getting-started/installation/databricks) | Teams already on Databricks | Databricks workspace | Medium |

:::tip Not sure which option is right for you?
[Book a call](https://calendly.com/prometheux/prometheux-intro?utm_source=docs)
with the Prometheux team and we'll help you choose the best deployment for your
use case.
:::
