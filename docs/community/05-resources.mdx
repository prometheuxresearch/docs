## 🤔 FAQ

### Common Questions

<details>
<summary>What's the difference between Prometheux and a graph database?</summary>

Prometheux is an ontology-native reasoning engine that goes beyond simple graph storage. It provides declarative rule-based reasoning, recursive query capabilities, and semantic understanding of your data.

</details>

<details>
<summary>Can I use Prometheux with my existing database?</summary>

Yes! Prometheux can integrate with various data sources including SQL databases, data lakes, and APIs. See our [integration guide](/learn/vadalog/data-sources).

</details>

<details>
<summary>What programming languages are supported?</summary>

Prometheux provides SDKs for Python, JavaScript/TypeScript, Java, and Go. The core reasoning is done in Vadalog, our declarative language.

</details>
